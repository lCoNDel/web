Creación de Skills y Funciones IA

Skill: TallerIA

Análisis predictivo de averías y optimización de stock de recambios basado en históricos.

// Skill creada en Antigravity
skill: analyze_repair_patterns
input: {modelo, horas_uso, historial}
output:
→ predict_failure_types()
→ recommend_stock_levels()
→ generate_work_order()
Integrado en Ollama Activo en WebUI

Función: StockOptimizer

Optimización automática de inventario multi-almacén con análisis de demanda en tiempo real.

// Function Antigravity
function: optimize_inventory
triggers: [sales_data, repair_requests]
actions:
• analyze_demand_patterns()
• calculate_optimal_stock()
• auto_generate_orders()
Ejecución automática Dashboard en WebUI

Connector: MercuryAPI

Conector directo con sistemas Mercury para actualización en tiempo real de datos técnicos.

// API Connector

• GET /parts/catalog
• POST /repair/manuals
• REALTIME /inventory/status
